# Shadow-IoT-Mapper

ShadowIoT is a red-team flavored network mapper for quick discovery of IoT devices.

## Setup

```bash
./setup.sh
```

## Usage

```bash
./shadowiot.sh
```

Menu options:

- Scan Network (mDNS/UPnP)
- Identify Vulnerable Printers
- Credential Audit (Check Default Passwords)
- Export Network Map (JSON)
